Árbol de Decisión

Un árbol de decisión divide los datos en ramas basadas en los atributos más informativos, hasta llegar a una predicción.

Pasos para construirlo:
  1. Calcular la entropía del conjunto de datos:
    2. $$ Entropía(S) = -p_+ \log_2(p_+) - p_- \log_2(p_-) $$
  2. Evaluar la ganancia de información para cada atributo:
    3. $$ Ganancia(S, A) = Entropía(S) - \sum_{v \in Valores(A)} \frac{|S_v|}{|S|} Entropía(S_v) $$
  3. Seleccionar el atributo con mayor ganancia y repetir el proceso por rama.

Si Edad = Joven y Estudiante = No ⇒ Compra PC = No
Si Edad = Viejo y Crédito Bueno = Excelente ⇒ Compra PC = No

Regresión Lineal

La regresión lineal encuentra una relación entre variables para hacer predicciones mediante una ecuación lineal:

$$ y = a + bx $$
  • x: variable independiente (ej. horas de estudio)
  • y: variable dependiente (ej. puntaje del examen)
  • a: intersección con el eje Y (constante)
  • b: pendiente de la recta

Para calcular a y b :

$$ b = \frac{n \sum xy - \sum x \sum y}{n \sum x^2 - (\sum x)^2} $$ $$ a = \frac{\sum y - b \sum x}{n} $$

Ejemplo con ecuación obtenida:

$$ y = 45 + 5x $$
Para x = 6: y = 45 + 5×6 = 75
Para x = 7: y = 45 + 5×7 = 80

Regresión Múltiple

Predice un valor y (ej. precio) en función de múltiples variables independientes:

$$ y = a + b_1 x_1 + b_2 x_2 $$
  • x₁: tamaño (Size)
  • x₂: número de habitaciones (Rooms)
  • a, b₁, b₂: coeficientes

Los coeficientes se calculan resolviendo:

$$ \vec{B} = (X^T X)^{-1} X^T Y $$

Ejemplo con ecuación obtenida:

$$ y = 50 + 0.9x_1 + 10x_2 $$
Para x₁=150 y x₂=5: y = 50 + 0.9×150 + 10×5 = 235

K-Medias (K-Means)

Agrupa datos en k clústeres y calcula la media de cada grupo para imputar valores faltantes:

  1. Agrupar datos por categoría (columna X0)
  2. Calcular la media de valores conocidos (X2) por grupo
  3. Reemplazar valores faltantes con la media del grupo
$$ \text{Media} = \frac{\sum \text{valores}}{\text{cantidad}} $$
Ejemplo:
Grupo A: 5, 7, 9 ⇒ Media = (5 + 7 + 9) / 3 = 7

Grupo B: 1, 11 ⇒ Media = (1 + 11) / 2 = 6

K-Modas (K-Modes)

Para datos categóricos, agrupa elementos similares y determina la moda (valor más frecuente) en cada grupo.

  1. Agrupar datos según Clase
  2. Identificar la moda en X1 por grupo
  3. Reemplazar ? con la moda del grupo
$$ \text{Moda} = \text{valor más frecuente en el grupo} $$
Ejemplo:
Clase = -: X1 = B, B, A ⇒ Moda = B
						
Clase = +: X1 = A, A, B, A ⇒ Moda = A
						
Se imputa ? - con B
